## วิธีการเริ่มต้นใช้งานเว็บไซต์
#### 1. เปิด Command Prompt
#### 2. เปลี่ยนไดเรกทอรีไปยังโฟลเดอร์ของโปรเจกต์โดยใช้คำสั่ง:
```bash
cd <folderของโปรเจค>
```
#### 3. รันโปรเจกต์โดยใช้คำสั่งต่อไปนี้ (สำหรับเบราว์เซอร์ Google Chrome):
```bash
run.bat
```
