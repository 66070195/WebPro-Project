## วิธีการเริ่มต้นใช้งานหน้าเว็ป
#### 1.เปิด Command Prompt ขึ้นมา
#### 2.ทำการเข้าถึงFolder โดยใช้คำสั่ง
```bash
cd <folderของโปรเจค>
```
#### 3.ทำการรันโปรเจคโดยพิมพ์คำสั่ง (Chrome)
```bash
run.bat
```
